1. Brand Promise and Key Message

"The Future of Finance Runs on Bitcoin."

Blockstream empowers individuals and institutions to secure, transact, and build on Bitcoin using open-source, verifiable, trust-minimized infrastructure.

Think

Bitcoin is the foundation for a fairer, more efficient global financial system. It's not just an asset for speculation.

Feel

Confident that funds are truly secure through self-custody and cryptographic verification, free from trust in third parties.

Do

Adopt Blockstream's products. Jade wallet for self-custody, Blockstream App for Bitcoin access, Liquid Network for faster transactions, and Blockstream Asset Management for institutional Bitcoin exposure.

Core Values

Permissionless innovation Don't trust, verify Open finance for everyone

2. Tone of Voice by Audience

Audience Tone Do Don't
Developers Direct, precise, technical Show code, link to repos, be honest about limitations Use marketing buzzwords, oversimplify, use metaphors
Bitcoiners Cypherpunk, principled, grounded Reference Bitcoin culture, emphasize self-custody and verification Use altcoin/DeFi hype language or references
Enterprise Professional, evidence-based, measured Lead with data, case studies, regulatory awareness Overpromise, ignore compliance concerns
Institutional Formal, structured, risk-aware Speak in portfolio theory, cite fund structures, reference credible names Use crypto slang, assume Bitcoin literacy, base on ideology

3. Main Challenges to Counter

"Nobody uses Liquid"

Counter with quarterly metrics reports, federation growth announcements, partner spotlights, and Liquid ecosystem maps. Let the data speak.

"Liquid is federated = centralized"

Push transparency around functionary operations. Tailor the message by audience: the maximalist purist can use it for small amounts and faster transactions; the enterprise user gets faster settlement and Confidential Transactions.

"All hardware wallets are the same"

Highlight what makes Jade genuinely different: fully open-source (hardware and firmware), air-gapped via QR codes, no proprietary secure element chip with potential backdoors, security disclosure transparency, and a lower price point than competitors.

4. Post Rewrite Example

Below is a real Blockstream post from X, followed by a proposed rewrite. The goal: same information, better structure for how people actually read on mobile.

Original post

Source: Blockstream on X — Simplicity / options post

Original
Bitcoin options infrastructure is being built without trusting an exchange, an oracle, or a counterparty. Simplicity enables on-chain options with high assurance smart contracting. No price feeds to manipulate. No smart contract exploits by design.
What could improve: The opening line is strong but the structure is flat. Four dense sentences with no visual hierarchy, so the reader's eye gets lost. The two most powerful lines ("No price feeds to manipulate. No smart contract exploits by design.") are buried at the end where most people stop reading. On mobile, this renders as a wall of text. No call to action, no link, no thread hook.

Proposed rewrite

B
Blockstream
@Blockstream
Bitcoin options. On-chain. No counterparty. Simplicity makes it possible: → No exchange to trust → No oracle to manipulate → No smart contract exploits: by design! High-assurance smart contracting, formally verifiable. Now live on Liquid. simplicity-lang.org
Why it works better: The first line is the hook — three words, three punches, stops the scroll. The vertical list makes each "no X" scannable on mobile. Same content, better spacing, stronger hierarchy.

Visual note

Post images across Blockstream's feed currently vary in style, layout, and color palette. A consistent card format — dark background, brand typeface for headings, one key visual element per post — would build instant recognition in the timeline, the same way a recurring column or segment works in editorial publishing.
